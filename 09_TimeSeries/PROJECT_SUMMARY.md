| `scipy` | 科学计算 | >= 1.7 |

### 实现的算法

**统计方法**:
- [x] Naive Forecast
- [x] Simple Moving Average (SMA)
- [x] Weighted Moving Average (WMA)
- [x] Single Exponential Smoothing (SES)
- [x] Double Exponential Smoothing (DES/Holt)
- [x] Triple Exponential Smoothing (TES/Holt-Winters)
- [x] ARIMA (p,d,q)
- [x] SARIMA (p,d,q)(P,D,Q)s

**机器学习方法**:
- [x] Random Forest Regressor
- [x] XGBoost Regressor

**诊断工具**:
- [x] ADF 检验（平稳性）
- [x] KPSS 检验（平稳性）
- [x] Ljung-Box 检验（残差白噪声）
- [x] ACF 分析
- [x] PACF 分析
- [x] 残差诊断（Q-Q图、直方图、时序图）

**评估指标**:
- [x] MAE (Mean Absolute Error)
- [x] RMSE (Root Mean Squared Error)
- [x] MAPE (Mean Absolute Percentage Error)
- [x] R² (Coefficient of Determination)
